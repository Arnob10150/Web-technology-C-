# Web Technologies Lab Tasks

This repository contains a collection of lab tasks covering HTML, CSS, and JavaScript fundamentals and advanced concepts for Web Technologies course supervised by Md.Khairul Alam Mazumder,lecturer, Faculty
DEPARTMENT OF COMPUTER SCIENCE,AIUB

## Project Structure

The repository is organized into three main directories:

### HTML/
Contains HTML lab tasks demonstrating various HTML elements and structures.
- **Lt-1/**: Basic HTML structure to **Lt-5/**: Advanced HTML features (includes Sports.html, World.html, Index.html)

### Css/
Contains CSS lab tasks focusing on styling and layout.
- **Lt-1/** to **Lt-10/**: Progressive CSS concepts from basic styling to advanced layouts

### Javascript/
Contains JavaScript lab tasks covering programming fundamentals and DOM manipulation.
- **Lt-01/** to **Lt-09/**: JavaScript concepts from variables and functions to advanced topics

## File Contents

Each lab folder typically includes:
- `index.html`: The main HTML file for the task
- `script.js`: JavaScript code (for JS labs)
- `style.css`: CSS styles (for CSS labs)
- `Lt-X.pdf`: Lab task description and requirements

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Arnob10150/Web-technology-C-.git
   ```

2. Navigate to any lab folder.

3. Open `index.html` in your web browser to view the task.

4. For JavaScript tasks, check the browser console for any outputs or errors.
